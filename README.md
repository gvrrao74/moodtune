# 🎵 MoodTune

**MoodTune** is a sleek, FastAPI-powered web application that curates the perfect Spotify playlist based on your current mood and preferred music genre. Whether you need high-energy Phonk for a workout or lo-fi beats for studying, MoodTune handles the vibe check.

## ✨ Features
* **Mood Selection:** Choose between Motivation, Chill, Focus, or Sadness.
* **Dynamic Genre Selection:** Pick from specialized genres like Hardstyle, Synthwave, and more.
* **Intensity Slider:** Fine-tune the energy levels (Visual UI element).
* **Live Spotify Embeds:** Listen to the recommended playlist directly within the app.
* **Mobile Responsive:** A clean, dark-themed UI that works on all devices.

## 🛠️ Tech Stack
* **Backend:** [FastAPI](https://fastapi.tiangolo.com/) (Python)
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Deployment:** [Render](https://render.com/)
* **Music Integration:** Spotify Official Editorial Playlists
